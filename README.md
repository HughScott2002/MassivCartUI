# MassivCartAPI

Backend API powering the **Massiv Cart platform**, a smart product price discovery and mapping system designed to help consumers quickly find the best prices across different retailers.

The API enables product search, price comparison, and store mapping functionality to support the Massiv Cart application.

---

## Project Overview

Massiv Cart is designed to solve the problem of **price transparency** in retail shopping.  

Consumers often visit multiple stores to compare prices manually.  
Massiv Cart simplifies this process by collecting and organizing product pricing information across retailers.

This API provides the backend services needed to power that experience.

---

## Key Features

- Product price comparison across retailers
- Retail store price mapping
- Product search and filtering
- Data aggregation for price analysis
- API endpoints for frontend applications
- Scalable backend architecture

## Core Objectives

The Massiv Cart API is designed to:

## Installation

Clone the repository:

```
git clone https://github.com/Jeamelee/MassivCartAPI.git
```

Navigate into the project folder:

```
cd MassivCartAPI
```

Install dependencies:

```
npm install
```

Start the server:

```
npm start
```
